# Proyecto de Lucas Kinderknech

## Información del Proyecto

Este proyecto forma parte del portfolio personal de **Lucas Kinderknech**, enfocado en el desarrollo fullstack moderno con tecnologías actuales.

El objetivo principal es crear una aplicación rápida, escalable y de diseño limpio, que sirva como base o demostración de buenas prácticas en desarrollo web.

## Tecnologías utilizadas

Este proyecto está desarrollado con:

* **Vite** – Herramienta de desarrollo ultrarrápida.
* **TypeScript** – Tipado estático para JavaScript moderno.
* **React** – Librería para construir interfaces de usuario.
* **shadcn/ui** – Componentes de interfaz accesibles y personalizables.
* **Tailwind CSS** – Framework CSS utilitario para un diseño ágil y responsivo.

## Cómo ejecutar el proyecto

Para ejecutar el proyecto localmente, asegurate de tener instalado **Node.js** y **npm** (se recomienda la instalación mediante [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

Sigue estos pasos:

```bash
# 1. Clonar el repositorio
git clone <URL_DE_TU_REPOSITORIO>

# 2. Ingresar al directorio del proyecto
cd <NOMBRE_DEL_PROYECTO>

# 3. Instalar dependencias
npm install

# 4. Iniciar el servidor de desarrollo
npm run dev
```

El proyecto se ejecutará con recarga automática y una vista previa instantánea en tu navegador.

## Cómo contribuir o modificar el código

Podés editar el código directamente desde tu entorno de desarrollo favorito o desde GitHub siguiendo estos pasos:

1. Navegá al archivo que quieras modificar.
2. Hacé clic en el ícono de **editar (✏️)**.
3. Realizá los cambios necesarios.
4. Commit y push para reflejar los cambios en el repositorio remoto.

También podés usar **GitHub Codespaces** para editar el proyecto en la nube:

1. En la página principal del repositorio, hacé clic en **Code → Codespaces**.
2. Creá un nuevo Codespace.
3. Realizá los cambios y confirmalos directamente desde ahí.

## Despliegue
